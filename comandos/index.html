Lista de comandos resumidos:

pagar-lançamento - Concluí um lançamento para a agência selecionada. Ver mais

pagar-missão - Concluí uma missão para a agência selecionada. Ver mais

mostrar-níveis - Mostra todos os níveis criados. Ver mais

remover-nível - Remove o nível selecionado. Redefine a ordem dos subsequentes. Ver mais

definir-prefíxo - Define um prefixo personalizado do bot para o servidor. Ver mais

adicionar-missão - Cria uma nova missão. Use "&" para separar argumentos. Ver mais

mostrar-missões - Mostra todas as missões criadas. Ver mais

remover-missão - Remove a missão selecionada e redefine a ordem das missões subsequentes. Ver mais

postar-missão - Posta no canal definido a missão selecionada. Ver mais

definir-canal-missões - Define o canal onde serão postadas as missões. Ver mais

definir-cargo-gerenciador - Define o cargo que pode usar alguns comandos específicos. Ver mais

registrar-agência - Registra uma agência para o usuário. Use & para separar argumentos. Ver mais

agência - Mostra informações sobre a agência do usuário.Ver mais

remover-agência - Remove permanentemente a agência do usuário.Ver mais

registros - Mostra os últimos dez registros de acontecimentos da agência.Ver mais

pesquisar - Faz um investimento em um determinado número de pacotes de pesquisa.Ver mais

atualizar - Avança para o próximo nível a agência do usuário.Ver mais

remover-dinheiro - Remove dinheiro da agência do usuário selecionado.Ver mais

adicionar-dinheiro - Adiciona dinheiro para agência do usuário selecionado.Ver mais

pagar - Transfere dinheiro para agência do usuário selecionado.Ver mais
